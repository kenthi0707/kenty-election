# 小林建登 立候補者紹介サイト

N高グループ 第5期生徒会役員選挙に向けた、小林建登さん（けんてぃー）の立候補者紹介サイトです。

中心メッセージは「知らないを、選べるに変える。」です。公約、活動経験、課題意識、質問・意見フォームへの導線を、スマートフォンで短時間に読める構成で整理しています。

## ファイル構成

- `index.html`: サイト本文とページ構造
- `style.css`: デザイン、レスポンシブ対応、アニメーション
- `script.js`: Q&Aアコーディオン、フォームURL設定、スクロール時の表示制御
- `README.md`: この説明ファイル

## GoogleフォームURLの差し替え方法

`script.js` の先頭にある次の行を、実際のGoogleフォームURLに変更してください。

```js
const FORM_URL = "https://example.com/form";
```

`index.html` 側のボタンには `data-form-link` が付いているため、上記を変えるだけで質問・意見フォームのリンク先が差し替わります。

## 画像差し替え方法

現在のファーストビューには、ノートとギターをモチーフにした仮ビジュアルを入れています。

本人写真や正式なビジュアルに差し替える場合は、`index.html` の `.portrait` 画像の `src` を差し替えてください。画像ファイルを使う場合は、例として `images/kobayashi.jpg` のように配置し、次のように変更します。

```html
<img class="portrait" src="images/kobayashi.jpg" alt="小林建登さんの写真" />
```

差し替え後は、必ず `alt` も画像内容に合わせて変更してください。

## ローカルでの確認方法

`index.html` をブラウザで開くと確認できます。

より本番に近い形で確認したい場合は、このフォルダで簡易サーバーを起動して確認してください。

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## 公開前チェックリスト

- GoogleフォームURLが本番用になっている
- 候補者名、肩書き、所属経験に誤りがない
- 本人写真や使用画像の掲載許可が取れている
- スマートフォンでファーストビューの文字が読める
- Q&Aがタップとキーボード操作で開閉できる
- 「投票してください」など選挙期間前に避けたい直接表現が入っていない
- 個人情報の扱いに関する注意書きがフォーム側にもある
- QRコードからアクセスしたとき、表示速度とリンク先が問題ない
